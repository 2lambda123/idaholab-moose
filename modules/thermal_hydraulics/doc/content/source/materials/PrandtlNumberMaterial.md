# PrandtlNumberMaterial

!syntax description /Materials/PrandtlNumberMaterial

The Prandtl number $Pr$ is computed as:

!equation
Pr = \dfrac{cp * \mu}{k}

with $cp$ the specific isobaric heat capacity, $\mu$ the dynamic viscosity and $k$ the thermal
conductivity.
