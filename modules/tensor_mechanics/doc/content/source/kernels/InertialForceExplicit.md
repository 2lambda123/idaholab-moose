# InertialForceExplicit

!syntax description /Kernels/InertialForceExplicit

## Description

This class computes the inertial force in weak form:
\begin{equation}
R(u_h) = (\psi_i, \rho \frac{\partial ^2 u_h}{\partial t ^2}) \quad \forall \psi_i,
\end{equation}
